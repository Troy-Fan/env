# Install

## Step 1:

Clone this repo at `~/` and rename to `.env`.

Modify `name` and `email` in `env/*/dotfiles/gitconfig` as your name and email.

## Step 2:

Execute the install script based your OS.

For MAC, execute:

    install_mac.sh

For Ubuntu, execute:

    install_ubuntu.sh

## Step 3:

Follow the instructions shown by the install script.
